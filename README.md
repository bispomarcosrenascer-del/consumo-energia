Calculadora de Consumo Elétrico

Este é um script simples desenvolvido em Python para auxiliar no controle doméstico de energia. O sistema permite que o usuário insira os dados de um eletrodoméstico e receba instantaneamente o cálculo do consumo mensal estimado em kWh.
O objetivo principal é promover a conscientização sobre o uso de energia elétrica e facilitar o cálculo de gastos fixos.

A Fórmula utilizada para o cálculo: consumo_mensal = (potencia * horas_dia * 30) / 1000
•	Potencia (W) → é o quanto o aparelho consome por hora, em watts 
•	horas dia → quantas horas por dia o aparelho fica ligado 
•	30 → número médio de dias no mês 
Potencia × horas dia × 30 = consumo total no mês em watt-hora (Wh).
Como a conta de energia usa quilowatt-hora (kWh) dividimos por 1000.

Instruções:
Digite o nome do aparelho:  tecle < enter >
Digite a potência do(a) aparelho em watts (W): tecle < enter >
Quantas horas por dia o(a) aparelho fica ligado(a)? tecle < enter >

O programa exibirá o resultado: 

Aparelho: xxxxx
Consumo estimado: xxxx kWh/mês
